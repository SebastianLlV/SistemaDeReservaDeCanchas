Sistema de Reserva de Canchas – Documentación del Proyecto

Este sitio web contiene la documentación completa del Proyecto de Reserva de Canchas. Aquí se incluyen las metodologías empleadas, el proceso de trabajo con Git y GitHub (inicialización, ramas, commits y pull requests), el historial del repositorio, así como la evidencia del despliegue en Netlify. También se presentan las contribuciones de los integrantes del equipo, capturas de pantalla de cada paso realizado y los enlaces al repositorio en GitHub y al sitio desplegado.
